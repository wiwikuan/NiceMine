# NiceMine 好雷 by NiceChord 好和弦

你被丟到一個 8x8 的遊戲區域裡的某處，裡面總共有 12 個地雷。你的目標是標示出全部的地雷，或是翻開全部沒有地雷的格子。

不過困難的點是，**在這遊戲中，你什麼都看不到**；需要全靠聽聲音來探索環境。

### 操作（小寫英文）：

- wsad 上下左右移動游標
- j 翻開格子
- k 標示（或解除標示）地雷
